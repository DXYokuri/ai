# Future UI Solar System Atlas

High-fidelity Future UI solar system atlas built with Vite, React, TypeScript, Three.js, GSAP, and shader-driven planet rendering.

## Local Commands

```bash
npm install
npm run dev
npm run build
npm test
```

The standalone HTML export is generated at `dist/atlas.html`.
